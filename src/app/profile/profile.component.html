Settings about Profile

To help decide when your boiler needs to start warming up we need some more information from you. If you know you will need water at a certain time, you can add a new entry in your usage profile. This entry needs a Name (this is the name of your boiler, default this is your first name.). A start time this is the time you want to start using the warm water (if you don't know a specific time use an estimate a little earlier than you think you'll need it.). An ending time this lets the algorithm know when you won't need the warm water anymore (If you need to estimate this estimate it a little longer than you expect to end using the warm water.). And finally a day, this determines what day of the week you will be needing warm water at the previously mentioned times.









Why this is important to us

With the help of this user profile we can adapt your boiler to be more efficient. The decision algorithm used to control your boiler will use this user profile to determine when to activate or when to deactivate. Thanks to this algorithm we can calculate when it is most efficient (and cheapest) to buy energy and activate your boiler. This will not only help the ennvironment but also save you money.
